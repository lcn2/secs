# secs

Print `seconds since the epoch` by the P1003.1 Posix/ISO standard method.


# To install

```sh
sudo make install
```


# Example

```sh
$ /usr/local/bin/secs
1745122888
```


# To use

```
/usr/local/bin/secs
```
